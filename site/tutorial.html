Introduction

Welcome to the Geddy.js tutorial. We'll walk you through getting started with Geddy, and in the process make a simple in memory to do list.

Installing Geddy

If you already have node.js and npm installed, installing Geddy is easy: 

npm install -g jake geddy

If you want to install from source, you can always git clone the repo and make install

git clone git://github.com/mde/geddy.git
cd geddy
make && sudo make install

Creating your first Geddy app

Make sure you're in the directory where you store all your code, and run:

geddy app todo_app

Running your first Geddy app

Similarly, use the geddy executable to start your app inside your app directory: 

cd todo_app
geddy

Viewing your app

To view your app, go to http://localhost:4000/ in your browser. You should see an app similar to this:

Todos Controller

We'll be doing most of our work in the todos.js controller

var Todos = function () {
  this.respondsWith = ['html', 'json', 'js', 'txt'];

  this.index = function (req, resp, params) {
    this.respond({params: params});
  };

  this.add = function (req, resp, params) {
    this.respond({params: params});
  };

  this.create = function (req, resp, params) {
    // Save the resource, then display index page
    this.redirect({controller: this.name});
  };

  this.show = function (req, resp, params) {
    this.respond({params: params});
  };

  this.edit = function (req, resp, params) {
    this.respond({params: params});
  };

  this.update = function (req, resp, params) {
    // Save the resource, then display the item page
    this.redirect({controller: this.name, id: params.id});
  };

  this.remove = function (req, resp, params) {
    this.respond({params: params});
  };

};

exports.Todos = Todos;
